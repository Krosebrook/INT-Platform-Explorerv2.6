# INTINC Persona KB (Mock)

This mock KB demonstrates persona extraction from INTINC's services and case studies.

## Files
- services_context.jsonl: Services with industries, roles, pain points, value props.
- work_context.jsonl: Case studies with problems, solutions, outcomes, metrics.
- persona_clues.jsonl: Atomic persona signals (goal, pain_point, success_metric).
- manifest/pages.json: Pages crawled.
- manifest/provenance.yaml: Crawl details.
- manifest/integrity.sha256: Checksums.

## ERD
services ↔ work_context (via service_slug) ↔ persona_clues (via role/industry).
